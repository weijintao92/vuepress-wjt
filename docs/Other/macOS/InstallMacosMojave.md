### 给苹果电脑装系统
1. 首先在 MacBook Air 关上的情况下，按着 Command 及 R 两个按键不放手，然后同时按下最右上角的开关键一下(记着此时仍然继续按着 Command + R 键)。
![An image](./1.jpg)

2. 选择第二项`Reinstall Mac OS X`

3. 选择`右上角`输入法为`简体拼音`，并连接wifi

4. 如出现：`无法连接到服务器 `的错误，一般都是时区的问题。
