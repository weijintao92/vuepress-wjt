### 面板概述
- 在一般的职场布线中，在每个工位上都会有一个网线插口的面板。我们在接这个面板的时候，需要根据面板的标准来接。（因为，面板极有可能并不是按照标准排的颜色）接完后，最后使用工具测试一下`线序`
