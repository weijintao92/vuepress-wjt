### 布隆过滤器原理（Bloom Filter）
- 数据库服务的第一层，用于过滤掉那些无效的请求。避免资源的浪费。
在布隆过滤器服务启动的时候，其会对现有的数据根据一定的规则进行解析。其会对，每个数据库请求进行一次过滤。只有，那些能够通过布隆过滤器的请求，才能进入到READIS的请求。
